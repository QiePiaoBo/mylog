# UserInfo

### 1. 添加角色

#### 请求方式

> POST

#### 请求路径

> /role

#### 请求报文

```json
{
  "pid": 0,
  "roleCode": "r1",
  "roleName": "admin",
  "roleDescription": "admin role",
  "roleSort": 1,
  "roleStatus": 1
}
```

#### 返回报文

```json
{
  "code": 0,
  "message": "成功",
  "data": {
    "id": 1,
    "pid": 0,
    "roleCode": "r1",
    "roleName": "admin",
    "roleDescription": "admin role",
    "roleSort": 1,
    "roleStatus": 1
  }
}
```
